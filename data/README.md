# Data Directory

This folder is intentionally **empty** in the public repository.

The dataset used in the analysis is **not included** here, as it is not publicly distributed with the code.  
To obtain the data, please contact the authors of the original study:

Daniela Di Cagno, Arianna Galliera, Werner Güth, Noemi Pace, and Luca Panaccione
"Experience and gender effects in acquisition experiment with value messages."
*Small Business Economics*, 48(1):71–97, 2017.
[https://doi.org/10.1007/s11187-016-9766-1](https://doi.org/10.1007/s11187-016-9766-1)

If you are a researcher and need access to the dataset for replication, please reach out to the authors directly.

Once the dataset is obtained, place the required files in this directory before running the analysis scripts.
